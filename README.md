# Neural-Nexus-The-Invincibles-
AI for Social Good 

#DATASET USED:
  Real time anomaly detection in CCTV surveillance 
  
#FEATURES:
  Detect and Classify Unusual events like violence, theft or accidents 
  Analysis real time CCTV footage 
  Detected events will be classified on a scale of 1-10 where 1 represents normal activity and 10 represents the most severe anomaly
  Notification will be sent to the security team to prioritize responses.
  
#TECHONOLOGIES USED:
  Kaggle Dataset 
  Python
